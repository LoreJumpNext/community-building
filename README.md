# introduction

LoreGit是一个刚刚发起的社区

我们希望能在社区聚集更多热爱开源与学术的伙伴，将git的开源方式拓展至学术领域，开源关于学术和知识的更多部分，建立一个更开放更专注于探索的泛开源社区

社区目前以LIB为核心载体，git作为版本控制系统。git本身略为复杂，不过可以简单的理解为公开的，具有协作能力的文件夹，这是一个非常重要的概念，起初可能稍微复杂，但是上手后就会发现它相当强大且高效。
